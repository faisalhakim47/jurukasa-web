# User Setup Flow
- user visit app
- app display welcome message
- app display information about the app
- app provide "Get Started" button
- user clicks "Get Started" button
- app navigates to user setup form
- app display user setup form with fields:
  - turso database url
  - turso database auth token
  - submit button
- user fill in turso database url
- user fill in turso database auth token
- user clicks submit button
- app validate turso database url and auth token
- if validation fails:
  - app display error message
  - app allow user to correct input
- if validation succeeds:
  - app save turso database url and auth token securely
  - app navigate to main application interface
- app display success message
